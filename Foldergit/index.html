Difference between HTML tags and HTML attributes

HTML tags HTML attributes
  1. HTML tags are used to hold the HTML element.
  2. HTML tags are almost like keywords where every single tag has unique meaning.
  3. HTML tag starts with < and ends with >
  4. For example:
    img,script,form,etc.
  1. HTML attributes are used to describe the characteristic of an HTML element in detail.
  2. HTML attributes specify various additional properties to the existing HTML element.
  3. HTML attributes are found only in the starting tag.
  4. For example:
    bgcolor,border,charset,etc.
